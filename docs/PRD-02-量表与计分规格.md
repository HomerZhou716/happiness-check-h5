# PRD-02 · 量表与计分规格

> 权威数据源 = `variants/paper/subjects.js`（导出 `HAPPINESS_DRIVING_DATA`）。本文件解释其结构与计分数学；两者不一致时以代码 + `app.test.js` 为准。**题干/选项文案改动会破坏规范哈希测试**，非任务明确要求勿改。

## 1. 量表来源与方法学

科目二/三的量表由用户原始 55 题《家庭幸福自检表》经三专家团队（心理测量 / 家庭关系 / 产品转化）论证、修订、合成而来，并经 Codex 两轮独立终审。核心方法学决策：

- **砍题**：55 → 24 计分题（+1 效标题）。原因四类——性别角色预设（如"谁拎包"，计分随答题者性别翻转）、社会赞许零区分度（"你是否重视 X"人人选最优）、条件题系统缺失（异地/双方父母类）、同构造重复（"支持兴趣"问了 4 遍）。
- **补题**：新增 6 题，填补家庭研究中预测力最强而原案空白的构面——冲突过程（Gottman 四骑士、和好复盘）、共同教养一致性（双教练是否拆台）、夫妻肢体亲密。
- **题目承接**：24 计分题中 17 道由原表改写、1 道由原表合并、6 道新增。
- **行为锚点**：每题三个选项是具体行为描述（非抽象"符合/不符合"），抗社会赞许，选项本身即传播素材。

> 上线期采用理论临时分档；累计 300+ 真实样本后应切换为百分位常模（见 ROADMAP）。3 点量表测量误差偏大，复检解读须遵守"分差 <8 分视为稳定"纪律。

## 2. 数据结构（`subjects.js`）

```
HAPPINESS_DRIVING_DATA = {
  subjects: { s2: {id,number,name,meta,dimension_codes}, s3: {…} },
  dimensions: [ {
      code, name, module,
      insight_low,                 // 该维度低分时的"点破"解读
      items: [ {
          id, text,
          options: [ {text, value} ],  // 3 项，value ∈ {2,1,0}，数组顺序即高→低分
          source,                       // 来源标注："原表题号 N 改写" / "新增(…)" / "…合并"
          deduction,                    // 0 分时的轻梗扣分文案

      } ]
  } ],
  plans:  { A1:{name,kicker,description,days:[…7]}, … B3 },  // 6 套 7 天陪练
  grades: [ {min,name,message} ],                             // 4 档，降序
  retest: [ {when,name} ],                                    // 4 条复检节奏
  stable_message
}
```

## 4. 计分数学

| 层级 | 公式 |
|---|---|
| 单题分 | 选中选项的 `value`：2 / 1 / 0 |
| 子维度分 | `round( 4题value合计 / 8 × 100 )` → 0–100 |
| 科目分 | `round( 3个子维度百分制分的均值 )` |
| 领证分 | `round( (科目二分 + 科目三分) / 2 )` |
| 扣分总数 | `100 − 领证分` |
| 报告扣分项 | 每个 0 分（C）题显示"扣 8 分"，形成补练清单；不改变上面的数值分 |

**防刷分**：三选项显示顺序逐题随机（`createOptionOrders`），随机序随进度持久化；语义 `value` 与显示位置解耦。

## 5. 危险驾驶红灯（4 项）

命中对应题的 0 分选项时点亮，独立于数值分数，体现"负性互动权重高于正性"（Gottman 5:1）：

| 维度 | 题 id | 行为 |
|---|---|---|
| A3 冲突与修复 | A3-2 | 吵架翻旧账、嘲讽或贬低 |
| A3 冲突与修复 | A3-3 | 冷战一周以上 |
| B2 尊重与回应 | B2-2 | 翻动孩子的手机、日记或房间 |
| B3 双教练协同 | B3-4 | 当孩子面争吵且不让其看到和好 |

## 6. 驾照等级（4 档，合规话术）

| 等级 | 领证分 | 话术要点（对行为不对人） |
|---|---|---|
| 教练级 | ≥85 | 开得很稳，有余力带别人 |
| 老司机 | ≥70 | 路况熟练，偶有小剐蹭；扣分项即补练清单 |
| 实习期 | ≥55 | 已上路，几个路口不熟；每周专练一项，人人都从实习期开始 |
| 陪练期 | <55 | 先别一个人开；和家人认真聊一次，可引入外部支持（父母成长课程、家庭教育指导） |

## 7. 复检机制

固定题本（不换题才能纵向比对）。四条节奏：3 个月「实习期考核」→ 6 个月「转正复检」→ 每年「幸福年审」→ 大事后「换证体检」。**解读纪律：两次分差 <8 分一律报"保持稳定"，不解读涨跌。**

## 8. 效标题（不计分）

原表第 55 题「整体满意度」改写保留为效标题，单列、不计入科目分，用于校验聚合分数的收敛效度（自评满意但得分低 = 触发"盲区提示"）。当前 H5 主要在配套 xlsx 版实现该对照；H5 端如落地见 ROADMAP。
