## 1. 一句话定位

用一次"驾照式"自检，帮已婚有孩的家长看清**夫妻这台车开得稳不稳、带娃这段路走得对不对**，并给出可执行的 7 天陪练——**不做泛泛的"测幸福"，只回答具体的关系痛点。**

## 2. 为什么是这个定位（战略结论）

项目经历过一次关键转向，结论来自独立的商业论证（Codex 红队 + 6 方向实证研究 + 三视角评审，详见内部《商业模式论证报告》）：

- **泛个人"测幸福"是弱需求、伪商业核心**——用户对"提升幸福感"的需求认可度仅约 27%、排第 5；真正的需求语言是"睡眠、解压、关系"。
- **家庭 / 亲子 / 夫妻关系是付费级痛点**——四方评审一致确认，关系定位显著强于泛幸福。
- 因此产品从「PERMA 个人幸福测评」转向「幸福驾校·家庭版」，个人测评降级为其中的"科目一"。

> 商业模式的完整判决、单位经济、90 天验证条件属敏感内容，见内部文档，不在本公开 PRD 展开。本 PRD 只覆盖**产品**层面。

## 3. 产品地图

```
幸福驾校（主入口 = 科目选择页）
├── 科目一 · 自我状态   PERMA 15题 · 7点量表 → 五维雷达报告 → 7天微训练 → 60秒呼吸  （沿用早期个人版）
├── 科目二 · 夫妻同行   12题 · A/B/C 行为锚点 → 科目报告 → 7天陪练
├── 科目三 · 亲子上路   12题 · A/B/C 行为锚点 → 科目报告 → 7天陪练
└── 幸福驾照            科目二+三均完成后可领 → 领证分/等级/扣分/复检路线（分享素材）
```

## 4. 目标用户

- **核心**：28–45 岁、孩子在学龄段的父母。
- **场景**：对夫妻关系或亲子教养有隐隐的不安，想要一次不带评判的"体检"和具体的下一步。
- **非目标**：临床人群、需要专业干预者——产品明确不承担诊断/治疗职能，触及严重问题时引导至外部支持。

## 5. 产品原则

1. **一屏一念**——移动端一屏只问一件事，题量控制在完成率安全区（单科约 4 分钟）。
2. **行为化，不评判**——题目问"你最近怎么做的"，不问"你是不是好父母"；等级话术"只对行为不对人"。
3. **看见即处方**——每个扣分项直接对应一条可执行的 7 天练习，不停留在"你有问题"。
4. **合规先行**——不触碰医疗用语；测评数据按敏感个人信息保守处理，默认仅存本地。
5. **东方静气**——纸境视觉（宣纸/墨/朱砂/描金），克制的动效，与"驾校"的轻松隐喻平衡。
