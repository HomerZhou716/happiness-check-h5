# 幸福自检 H5 Demo

一个零依赖、移动优先的纯静态体验：从 15 题 PERMA 自检进入五维报告，再按当前弱项生成 7 天微训练；Day 1 提供可实际完成的 60 秒呼吸引导。所有图形均由 CSS 或内联 SVG 绘制，页面无需联网。

## 如何打开

直接双击 `index.html`，或将它拖入任意现代浏览器即可运行。桌面浏览器会以 430px 限宽居中展示，推荐同时用 375–430px 宽度查看移动端效果。

## 数据结构

题库、维度文案、量表标签、计分规则、推荐映射与三套训练计划集中在 `app.js` 顶部。答题记录以 15 位数组保存，每项为 `1–7` 或 `null`；完成后生成 `{ total, scores }`，其中 `scores` 包含 `P / E / R / M / A` 五项 `0–100` 分值。进度、结果、推荐方向与 Day 1 完成状态保存在浏览器 `localStorage` 的 `happiness-self-check:v1` 条目中。
